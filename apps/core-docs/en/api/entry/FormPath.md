# FormPath

Please refer to the [`@silver-formily/path` documentation](https://path.silver-formily.org/). `@silver-formily/core` fully reuses and re-exports its internal `Path` class.

::: warning Note
If you import from `@silver-formily/path`, the class name is `Path`. If you import from `@silver-formily/core`, the class name is `FormPath`. They are equivalent, but the exported names are different.
:::
