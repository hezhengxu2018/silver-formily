## API

See [https://element-plus.org/en-US/component/input-tag.html](https://element-plus.org/en-US/component/input-tag.html)

### Slots

| Slot     | Description                                            | Type                               |
| -------- | ------------------------------------------------------ | ---------------------------------- |
| `prefix` | Content shown before the input                         | --                                 |
| `suffix` | Content shown after the input                          | --                                 |
| `tag`    | Custom tag renderer with an injected `field` reference | ^[object]`{ value, index, field }` |
