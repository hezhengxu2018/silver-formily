# Silver Formily Vant

`@silver-formily/vant` 现在已经有一版最小可运行骨架，当前可以先体验 `FormItem` 和 `Input` 的组合效果。

## 当前可用

- `FormItem`
- `Input`
- `Input.TextArea`
- `PreviewText.Input`

## 快速入口

- [快速开始](/component/quick-start)
- [FormItem 示例](/component/form-item)
- [Input 示例](/component/input)

## 说明

当前阶段仍然是基础封装优先，先把 Vant 和 Formily 的职责边界梳顺，再继续补更多移动端组件。
