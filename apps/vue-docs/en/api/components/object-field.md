# ObjectField

## Description

Vue binding for [createObjectField](https://core.silver-formily.org/en/api/models/Form#createobjectfield) in `@silver-formily/core`. It links object-shaped values to layout or input components, and follows [IFieldFactoryProps](https://core.silver-formily.org/en/api/models/Form#ifieldfactoryprops).

The `name` prop is required. Compose child components through scoped slots.
