# @silver-formily/json-schema

Silver Formily 对 `@formily/json-schema` 的本地维护版本。

## 安装

```bash
pnpm add @silver-formily/json-schema
```

## 说明

- 公开 API 与上游 `@formily/json-schema` 保持兼容。
- 仓库内依赖已切换到 `@silver-formily/core`、`@silver-formily/reactive`、`@silver-formily/shared`。
- 详细协议文档见 <https://json-schema.silver-formily.org>。
