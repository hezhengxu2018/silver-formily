# Shared Package Guide (extends root `AGENTS.md`)

This package is the locally maintained fork of `@formily/shared`. Treat it as foundational infrastructure for the other Silver Formily runtime packages.

## Source Layout

- `src/path.ts`, `src/checkers.ts`, `src/merge.ts`, and related helpers should stay small and dependency-light.
- `src/__tests__` preserves upstream utility coverage; keep additions targeted to utility regressions.
- Build output goes to `esm/` and is generated by tsdown.

## Verification

- Run `pnpm --filter @silver-formily/shared build` after changing helpers used by other packages.
- Prefer validating downstream consumers after touching path utilities or merge semantics.
